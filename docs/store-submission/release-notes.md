# Release Notes Draft

## Store Candidate Draft

Blockiverse VR is preparing its first Meta Quest store candidate.

This candidate includes:

- Quest controller VR play with comfort-focused locomotion.
- Creative voxel block placement and removal.
- Generated voxel terrain using original authored block textures.
- Survival-lite resource gathering, inventory, crafting, storage, and health systems.
- Local save/load persistence for world and player state.
- Local LAN co-op with host-authoritative world state.
- Meta Horizon Avatar integration where platform avatar support is active, with fallback representation for validation.
- Original Blockiverse VR branding, icon, launch, texture, item, and UI assets.
- Local diagnostics support for validation and issue investigation.

Not included in this candidate:

- Public matchmaking.
- Public community worlds.
- Cloud-hosted persistent worlds.
- In-app voice chat.
- In-app text chat.
- Ads or in-app purchases.

Final release notes must be updated after signed build creation, Quest 3/3S validation, performance capture, and Eric's submission review.
