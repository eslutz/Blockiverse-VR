# Blockiverse VR - Privacy Policy Source

The maintained public privacy policy for Blockiverse VR is:

https://blockiversevr.com/privacy/

Use that URL in Meta Developer Dashboard privacy policy fields and store review materials.
The website page is the only maintained privacy policy text; do not duplicate the policy body
in this repository.

## Store submission checklist note

- Confirm `https://blockiversevr.com/privacy/` is live over HTTPS before submission.
- Confirm the public page still discloses Meta platform data, user age category handling,
  local game data, LAN multiplayer data, diagnostic logs, no advertising or analytics SDKs,
  and deletion request instructions.
- Update the website privacy page first whenever shipped behavior changes.
