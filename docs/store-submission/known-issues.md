# Known Issues Draft

This draft is for store-candidate review notes. It should be updated after final Quest 3 and Quest 3S smoke tests.

## Current Candidate Notes

- The first store candidate is scoped to single-player and local LAN co-op. Public matchmaking, cloud-hosted worlds, invite-code cloud worlds, and community discovery are not included.
- Voice chat is not built into Blockiverse VR. Players can use Meta Quest party chat outside the app.
- Store-candidate performance evidence is still pending final OVR Metrics capture on Quest 3 and Quest 3S.
- Final store screenshots and artwork are planned but have not been uploaded externally.
- A public privacy policy URL and public support path must be published before real submission.

## Not Known To Be Shipped In This Candidate

- In-app text chat.
- In-app voice capture or voice transport.
- Ads.
- In-app purchases.
- Public user-generated text.
- Cloud persistence.
- Custom Blockiverse account system.
