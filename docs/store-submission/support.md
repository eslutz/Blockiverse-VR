# Support And Public Links Draft

This file tracks support information needed before external submission. No public link is created as part of this in-repo prep pass.

## Publisher

Eric Slutz

## Support Scope

Support should cover:

- Install and launch issues.
- Save/load problems.
- Quest 3 or Quest 3S compatibility issues.
- Comfort or input issues.
- Privacy questions and deletion requests for any data controlled outside the player's device.
- Store-candidate known issues and release notes.

## Required Before Submission

- Select a public support URL or support email.
- Publish the privacy policy at an HTTPS URL.
- Enter the privacy policy URL in the Meta Developer Dashboard.
- Enter the support URL or support contact in the Meta Developer Dashboard.
- Confirm the chosen support path can receive deletion and privacy requests.

## In-Repo Status

The repository contains draft support text only. External hosting, dashboard fields, and public support routing remain out of scope for this work.
